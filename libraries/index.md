**Stanix** comes with multiples system libraries.  
These libraries include :  
- [tlibc](tlibc)
- [libgfx](libgfx)
- libterm
- [libtwm](libtwm)
- libinput
- libtrm

Others libraries can be installed on the system, check the docs specific the the libraries for more info about them.
