## use
A `vfs_fd_t` represent an **open context** with a file or a device.

## operations
A list of operation that can be used on `vfs_fd_t`.
- [`vfs_dup`](vfs_dup)
- [`vfs_close`](vfs_close)
- [`vfs_read`](vfs_read)
- [`vfs_write`](vfs_write)
- [`vfs_ioctl`](vfs_ioctl)
- [`vfs_mmap`](vfs_mmap)
