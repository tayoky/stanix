## description
**twm (Tayoky's Window Manager)** is the window manager of **stanix**. 
**twm** is the server while client can connect to **twm** using [libtwm](../../libraries/libtwm).

## files
`/etc/twmrc` is executed at startup of **twm**.  
Images and themes for use in **twm** are stored in `/usr/share/twm`.
