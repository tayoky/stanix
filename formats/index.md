**Stanix** uses a lot file formats, some are standard others are not. Only non standard file formats are explain in details.

- ELF binaries
- QOI images
- [Stanix's libinput keyboard layout](layouts)
- C files
- AR archives
- TAR archives
- shell scripts
- linker scripts
- makefiles
